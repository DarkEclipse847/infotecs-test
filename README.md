# Тестовое для стажировки в infotecs

Использовал только один запрос, дабы избежать ошибки 429
Решил не переизобретать велосипед, а воспользоваться tanstack table(бывшая react-table) - легковесной библиотечкой для работы с таблицами
Воспользовался архитектурой FSD(чуть-чуть надо подправить, но иерархия и хранение компонентов соблюдены)

### Сделано:

- Изменения размера колонок(минимум 50px) -> потянуть за синюю штучку у хедера колонки
- Сортировка по полям -> тыкнуть на хедер колонки
- Фильтрация(поиск) -> строка поиска в хедере
- Кликабельные данные с выводом модального окна
- Ширина таблицы 100% от viewport, до 1400px
- Современный подход к строению приложения
